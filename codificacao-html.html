    Tag a

    Tag (a) define um hiperlink, que é usado para vincular de uma página a outra

                
                <a href="https://www.exemplo.com">Visite nosso site</a>

    Tag img

    A tag (img) é usada para incorporar uma imagem em uma página HTML. As imagens não são
    tecnicamente inseridas em uma página da web, as imagens estão vinculadas a páginas da web. A img tag
    cria um espaço de retenção para a imagem referenciada.A img tag tem dois atributos obrigatórios:
    src - Especifica o caminho para a imagem
    alt - Especifica um texto alternativo para a imagem, se a imagem por algum motivo não puder ser exibida

                
                <img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

    Tag p

    Tag (p) define um parágrafo

                
                <p>This is some text in a paragraph.</p>

    Tag div

    A tag (div) define uma divisão ou seção em um documento HTML. A div tag é usada como um
    contêiner para elementos HTML, que são então estilizados com CSS ou manipulados com JavaScript. A div tag é facilmente estilizada usando o atributo class ou id.

        
            
                <div class="myDiv">
                    <h2>This is a heading in a div element.</h2>
                    <p>
                        This is some text in a div element.
                    </p>
                </div>

    Tag li

    Tag (li) define um item da lista

                
                <li> Coffee </li>
                <li> Tea </li>
                <li> Milk </li>

    Tag (ul) define uma lista não ordenada com marcadores

                
                <ul>

    Tag span

    tag (span) é usado para colorir uma parte de um texto

                
                <p>
                    My mother has <span style="color:blue"> blue </span> eyes.
                </p>

    Tag header

    tag (header) representa um contêiner para conteúdo introdutório ou um conjunto de links
    de navegação. Um header elemento normalmente contém: um ou mais elementos de título ( h1 - h6 ),
    logotipo ou ícone, informações de autoria

                
                <header> 
                    <h1>A heading here</h1> 
                    <p>Posted by John Doe</p> 
                    <p>Some additional information here</p> 
                </header>

    Tag footer

    tag (footer) define um rodapé para um documento ou seção

                
                <footer>
                    <p> Author: Hege Refsnes </p> 
                    <p> <a href="mailto:hege@example.com">hege@example.com </a> </p>
                </footer>

    Tag section

    tag (section) define uma seção em um documento podendo ter mais seções

                
                <section>
                    <h2> WWF History </h2> 
                    <p> 
                        The World Wide Fund for Nature (WWF) is an international organization 
                        working on issues regarding the conservation, research and restoration 
                        of the environment, formerly named the World Wildlife Fund. WWF was founded in 1961.
                    </p>
                </section>

    Tag article

    tag (article) especifica conteúdo independente. Um artigo deve fazer sentido por si só
    e deve ser possível distribuí-lo de forma independente do resto do site.

                
                <article>
                    <h2 Google Chrome </h2>
                    <p> 
                        Google Chrome is a web browser developed by Google, released in 2008. Chrome is the 
                        world's most popular web browser today! 
                    </p>
                </article>

    Tag nav

    tag (nav) define um conjunto de links de navegação.

                
                <nav>
                    <a href="/html/">HTML </a> 
                    <a href="/css/">CSS </a> 
                    <a href="/js/">JavaScript </a> 
                    <a href="/python/">Python </a>
                </nav>
